
# 🍽️ PiringNegeri
**PiringNegeri** is an ecosystem dedicated to Indonesian culinary heritage, connecting food enthusiasts with authentic flavors through modern technology.

---

## 🏗️ Project Architecture
Our ecosystem is built with a focus on performance, scalability, and seamless user experience across all platforms:

* **Backend (Core API):** [piringnegeri-API](https://github.com/PiringNegeri/piringnegeri-API) - A robust Node.js-based REST API serving as the central nervous system for data and business logic.
* **Frontend (Web):** **Astro** - Leveraging the island architecture to build a lightning-fast, SEO-optimized web platform for culinary discovery.
* **Mobile (Android/iOS):** **React Native** - Delivering a high-performance native mobile experience for users on the go.

## 📋 Getting Started

To set up the backend API locally:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/PiringNegeri/piringnegeri-API.git
    ```
2.  **Install Dependencies:**
    ```bash
    cd piringnegeri-API
    npm install
    ```
3.  **Environment Setup:**
    Copy `.env.example` to `.env` and fill in your local configuration credentials.
4.  **Run Development Server:**
    ```bash
    npm run dev
    ```
